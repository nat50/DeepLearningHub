📚 Text Preprocessing Pipeline

Step 1: Lowercase

Convert all text to lowercase for consistency — "Cat" and "cat" become the same token.

Step 2: Remove URLs & HTML

Strip http/https links and HTML tags — they carry no semantic meaning for topic classification.

Step 3: Remove Punctuation

Keep only alphanumeric characters and spaces — reduces vocabulary size without losing meaning.

Step 4: Normalize Whitespace

Collapse multiple spaces and trim — produces consistent token boundaries.

Impact:

Reduces total words from 128M to 57M (55% reduction) while preserving semantic content.

import re

def clean_text(text):
    if not isinstance(text, str):
        return ""
    text = text.lower()
    text = text.replace('\\n', ' ').replace('\\r', ' ').replace('\\t', ' ')
    text = re.sub(r'https?://\S+|www\.\S+', '', text)
    text = re.sub(r'<[^>]+>', '', text)
    text = re.sub(r'[^\w\s]', '', text)
    text = re.sub(r'\s+', ' ', text).strip()
    return text

🧠 LSTM Architecture

▼

📚 Bidirectional LSTM Classifier

Embedding Layer:

Embedding(50,002, 128) — maps integer tokens to 128-dimensional dense vectors. Trained from scratch.

Bidirectional LSTM:

2-layer Bi-LSTM with 256 hidden units per direction. Bidirectional means it reads text left→right AND right→left, capturing context from both directions. Hidden state: 256 × 2 = 512 dimensions.

Classification Head:

Concatenate final forward & backward hidden states → Dropout(0.3) → FC(512 → 10)

Pack Padded Sequences:

Variable-length inputs are packed so the LSTM ignores padding tokens — more efficient and accurate.

import torch.nn as nn

class myLSTM(nn.Module):
    def __init__(self, vocab_size, embedding_dim=128, hidden_dim=256,
                 output_dim=10, n_layers=2, bidirectional=True,
                 dropout=0.3, pad_idx=0):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embedding_dim, padding_idx=pad_idx)
        self.lstm = nn.LSTM(
            embedding_dim, hidden_dim,
            num_layers=n_layers, bidirectional=bidirectional,
            dropout=dropout if n_layers > 1 else 0.0, batch_first=True,
        )
        direction_factor = 2 if bidirectional else 1
        self.fc = nn.Linear(hidden_dim * direction_factor, output_dim)
        self.dropout = nn.Dropout(dropout)

    def forward(self, text, lengths):
        embedded = self.dropout(self.embedding(text))
        packed = nn.utils.rnn.pack_padded_sequence(
            embedded, lengths.cpu().clamp(min=1),
            batch_first=True, enforce_sorted=False)
        _, (hidden, _) = self.lstm(packed)
        combined = torch.cat([hidden[-2], hidden[-1]], dim=1)
        return self.fc(self.dropout(combined))

🤖 BERT Transformer Architecture

▼

📚 BERT Base Uncased Classifier

Backbone:

bert-base-uncased — 12 Transformer layers, 768 hidden dimensions, 12 attention heads, ~110M parameters. Pre-trained on English Wikipedia + BookCorpus.

Tokenizer:

WordPiece tokenizer (30,522 vocab). Handles out-of-vocabulary words by splitting into subword units (e.g., "playing" → ["play", "##ing"]).

Classification Head:

[CLS] token → Dropout(0.3) → FC(768 → 10). The [CLS] token captures the entire sequence representation.

Fine-tuning:

All BERT parameters updated (no backbone freezing). AdamW optimizer with lr=2e-5, the standard BERT fine-tuning recipe.

import torch.nn as nn
from transformers import AutoModel, AutoConfig

class myTransformer(nn.Module):
    def __init__(self, checkpoint="bert-base-uncased",
                 num_classes=10, dropout_rate=0.3, freeze_backbone=False):
        super().__init__()
        config = AutoConfig.from_pretrained(checkpoint)
        self.bert = AutoModel.from_pretrained(checkpoint, config=config)
        self.dropout = nn.Dropout(dropout_rate)
        self.classifier = nn.Linear(config.hidden_size, num_classes)
        if freeze_backbone:
            for param in self.bert.parameters():
                param.requires_grad = False

    def forward(self, input_ids, attention_mask=None):
        outputs = self.bert(input_ids=input_ids, attention_mask=attention_mask)
        cls_output = outputs.last_hidden_state[:, 0, :]
        return self.classifier(self.dropout(cls_output))
